# User Journeys

## 1. Business Onboarding
Sign up → create business profile → set timezone/currency/service area → create first service → set availability → receive booking page URL.
**Done:** a working public booking page exists after onboarding.

## 2. Customer Booking Request
Open booking page → select service → select day/time → enter contact/address/notes → submit → see confirmation → business receives request.
**Done:** request appears in dashboard with all details.

## 3. Booking → Job
Owner opens request → reviews → accepts/edits time → customer created if needed → job created → confirmation sent.
**Done:** job appears in calendar and customer history.

## 4. Team Job Execution
Crew opens mobile view → sees today's jobs → opens job → reads notes/checklist → marks in progress → completes checklist → uploads photos → marks complete → customer gets completion email.
**Done:** job status and activity log update correctly.

## 5. Payment Tracking
Owner opens completed job → sends payment link or marks status → customer pays / owner records → dashboard outstanding updates.
**Done:** payment status visible on job, customer, payments page, dashboard.
