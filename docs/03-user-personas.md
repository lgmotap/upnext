# User Personas

## 1. Solo Operator (priority: very high)
Independent cleaner/handyman/pet walker/mobile detailer. Needs to look professional without admin staff, avoid lost bookings, run jobs from phone, track who paid, keep notes organized.

## 2. Growing Service Team (very high)
Cleaning company with 3–15 members. Needs to assign jobs, see availability, keep checklists consistent, avoid confusion, track work and revenue.

## 3. Operations Manager (high)
Office/admin person. Needs booking inbox, calendar, customer records, payment follow-up, reminders, status.

## 4. Field Worker (high)
Cleaner/gardener/painter/dog walker. Needs today's jobs, address/time, customer notes, checklist, photo upload, mark done.

## 5. End Customer (high)
Homeowner/office manager. Needs easy booking, clear pricing, confirmation, reminder, receipt/completion summary.
